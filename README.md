# Лабораторная работа 6 — SPA без фреймворков

Приложение построено на чистом JavaScript и реализует SPA с несколькими экранами:

- `#users` — список пользователей;
- `#users#todos` / `#users#ID#todos` — задачи всех пользователей или конкретного;
- `#users#posts` / `#users#ID#posts` — посты;
- `#users#ID#posts#POST_ID#comments` — комментарии к посту;
- `#add-user` — форма добавления локального пользователя и его задач (сохраняются в `localStorage`).

## Запуск

1. Откройте `index.html` через простой HTTP-сервер, например:
   ```bash
   npx serve .
   ```
   или запустите через Live Server / встроенный в редактор статический сервер.
2. Перейдите по адресу `http://localhost:5000` (порт зависит от выбранного сервера).

## Архитектура

- `index.html` — единственный HTML с корневым элементом `#app`.
- `js/utils.js` — хелперы (`createElement`, `debounce`).
- `js/router.js` — простейший роутер на hash.
- `js/components/*` — переиспользуемые компоненты (строка поиска, breadcrumbs).
- `js/screens/*` — экраны приложения.
- `js/api.js` — работа с JSONPlaceholder + локальные данные.
- `style.css` — стилизация карточек, форм, кнопок.

## Требования лабораторной

- Без фреймворков, препроцессоров и бандлеров.
- Хлебные крошки, поиск с дебаунсом, добавление/удаление локальных пользователей.
- Красивые стили и грамотная структура проекта.
- Командная работа через git (ветки, понятные коммиты, пулреквесты).
